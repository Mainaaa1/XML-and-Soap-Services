---

## What is JAX-WS?

**JAX-WS (Java API for XML Web Services)** is the standard Java framework used to create and consume SOAP web services.

What I found useful about it:

* It can automatically generate client code from a **WSDL**
* It handles XML serialization and deserialization
* It sends SOAP requests without me manually building XML

Essentially, it allows SOAP services to be consumed as if they were normal Java libraries.

## Step 1 – Example SOAP Service

For practice, I used a public SOAP calculator service:

```
http://www.dneonline.com/calculator.asmx?WSDL
```

This service exposes several operations:

* Add
* Subtract
* Multiply
* Divide

---

## Step 2 – Generate Client Classes

Java provides a command-line tool called **wsimport** that generates Java classes from a WSDL.

I ran:

```
wsimport -keep http://www.dneonline.com/calculator.asmx?WSDL
```

This generated several Java classes including:

* Service classes
* Request/response models
* SOAP bindings

These generated files allow the SOAP API to be used like a normal Java dependency.

---

## Step 3 – Java SOAP Client Example

After generating the classes, I created a simple client program.

```java
import com.dneonline.www.Calculator;
import com.dneonline.www.CalculatorSoap;

public class SoapClientExample {

    public static void main(String[] args) {

        Calculator service = new Calculator();
        CalculatorSoap soap = service.getCalculatorSoap();

        int result = soap.add(5, 3);

        System.out.println("Result: " + result);
    }
}
```

---

## What Happens Internally

When I call:

```
soap.add(5,3)
```

Java automatically:

1. Builds a SOAP XML request
2. Sends an HTTP request to the SOAP endpoint
3. Receives the SOAP XML response
4. Converts the XML response back into Java objects

Example SOAP request generated by the client:

```xml
<soap:Envelope>
  <soap:Body>
    <Add>
      <intA>5</intA>
      <intB>3</intB>
    </Add>
  </soap:Body>
</soap:Envelope>
```

## Step 4 – Compile and Run

To test the client I compiled and ran the program.

Compile:

```

```

Run:

```

```
